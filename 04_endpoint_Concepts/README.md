# 04 Endpoint Concepts

この章では、「エンドポイント（endpoint）」という言葉を  
**できるだけ広い意味から**  整理します。

前章（03_FastAPI_Basics）で FastAPI を実際に動かした経験をもとに、
言葉としての「エンドポイント」を理解することが目的です。

---

## エンドポイントとは何か（広い意味）

エンドポイントとは、

> **外部からアクセスするための「入口」**

を指す言葉です。

- 人がアクセスする入口
- プログラムがアクセスする入口
- ネットワーク越しの入口

このように、「何かを使うための到達点」を  
まとめて **エンドポイント** と呼びます。

---

## Web におけるエンドポイント

Web の世界では、エンドポイントは次のように表現されます。

```text
http://127.0.0.1:8000/
```
これは、

http:// : 通信方法
127.0.0.1 : 接続先
:8000 : ポート番号
/ : パス

を組み合わせた **アクセス先（入口）** です。
-----
## FastAPI におけるエンドポイント

FastAPI では、エンドポイントは コードとして定義されます。

```python
@app.get("/")
def read_root():
    return {"message": "Hello World"}
```

ここでは、

- "/" がエンドポイント
- GET メソッドでアクセス可能
- この URL にアクセスすると関数が呼び出される

という関係になっています。
